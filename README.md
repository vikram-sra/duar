# DUAR - Explore the Dimensions

A minimal, cinematic 3D immersive web experience built with **Three.js** and **GSAP**. Users explore a mysterious landscape containing portals (doors) that lead to various sections like Portfolio, Blog, and Projects.

![Duar Screenshot](https://via.placeholder.com/800x450?text=Duar+Experience)

## Features

*   **Immersive 3D Environment**: A clean, minimal atmospheric scene with dynamic lighting.
*   **Cinematic Lighting**: Real-time Day/Night cycle elements with a fiery, glowing sun and a cool, distant moon.
*   **Interactive Portals**: Distinct, animated doors that open interactively to reveal destinations.
*   **Performance Optimized**: Tuned for performance with efficient rendering settings, optimized geometries, and resource management.
*   **Responsive Control**: Full 360-degree camera orbital controls with ground collision prevention.

## Tech Stack

*   **Three.js**: Core 3D engine for rendering the scene, lights, and models.
*   **GSAP (GreenSock)**: Smooth, high-performance animations for camera movements and door interactions.
*   **Vite**: Fast development build tool.

## Installation & Running Locally

1.  Clone the repository:
    ```bash
    git clone https://github.com/vikram-sra/duar.git
    cd duar
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser at `http://localhost:5173`.
